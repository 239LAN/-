服务器规则

请所有玩家遵守以下规则，共同维护良好的游戏环境

禁止行为

  • 禁止使用任何外挂、作弊客户端或利用游戏漏洞
  • 禁止恶意破坏或偷盗其他玩家的建筑或财产
  • 禁止在聊天栏刷屏、发布广告或进行人身攻击
  • 禁止使用不文明ID或皮肤
  • 禁止恶意PVP

建议行为

  • 友好对待新玩家，主动提供帮助
  • 主动保护自己的财产，创建领地保护自己的建筑
  • 在建筑时考虑周围环境，保持美观
  • 发现BUG及时向管理报告
  • 公共建筑请标注建造者和用途

处罚规则

  • 首次违规：视严重情况予以警告或直接封禁等处罚
  • 二次违规：禁言或封禁1-7天等处罚
  • 三次以上违规：永久封禁或其他处罚
  • 使用外挂：永久封禁，不予申诉

关于我们

了解创想世界服务器的创建理念和联系方式

服务器历史

创想世界服务器成立于2024年10月26日，由一个热爱Minecraft的玩家创建。我们的目标是打造一个友好、公平、有趣的Minecraft生存服务器，让玩家能够享受原版生存的乐趣，同时也能体验到社区互动的温暖。

服务器基于Minecraft Java版 1.21.11的Leaf服务端，在保持原版特性的同时，优化了游戏性能，提供了更稳定的游戏环境。

加入服务器

要加入我们的服务器，请按照以下步骤操作：

  1. 确保您的Minecraft版本为1.21.11
  2. 启动Minecraft Java Edition
  3. 点击"多人游戏"，然后点击"添加服务器"
  4. 在服务器地址中输入：cxsjmc.cn
  5. 点击"完成"，然后选择我们的服务器并点击"加入服务器"

提示：如果无法连接，请检查您的网络设置或联系管理员，你也可以在服务器状态页面查询服务器在线状态。
